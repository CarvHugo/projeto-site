# Projeto — Página Web de Restaurante

Este projeto tem como objetivo evoluir uma aplicação web de restaurante, integrando progressivamente conceitos de Front-End e Back-End conforme avanço nos estudos.

Atualmente estou estudando HTML5, CSS e SQL, com foco em desenvolvimento web. A proposta é transformar este site em uma aplicação cada vez mais próxima da realidade de um negócio, incorporando funcionalidades como banco de dados, autenticação e gerenciamento de informações.

Já desenvolvi anteriormente um projeto de [cardápio virtual](https://github.com/CarvHugo/pratica_cardapio) em Python, cuja ideia servirá como base para o Back-End desta aplicação.
